# CADBIM Web Sitesi

Statik HTML sitesi. Tüm sayfalar `site/` klasöründe.

- Giriş: `site/index.html`
- Paylaşılan nav/menü mantığı: `site/mobilenav.js`
- Varlıklar (css/img/logo): `site/assets/`
- Diğer tüm sayfalar (`cadbim_*.html`, `sektor_*.html`, KVKK sayfaları vb.) aynı klasörde, düz yapıda.

## GitHub
Bu proje `bzkindamix/cadbim_website` (main) reposuyla eşleşir. Repo kökünde dosyalar `site/` ön eki olmadan aynı düz yapıda durur (`index.html`, `mobilenav.js`, `cadbim_*.html`, `assets/`).

Claude Code veya başka bir editörde açarken `site/` klasörünü kök olarak kullanın; kod düzenlemeleri buradan yapılıp GitHub reposuna o şekilde taşınmalı.

## Not
`uploads/` klasöründeki eski/yinelenen kopyalar kaldırıldı — güncel ve tek doğru kaynak `site/` klasörüdür.
