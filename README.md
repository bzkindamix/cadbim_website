# 🌐 CADBİM Website

CADBİM'in kurumsal web sitesi — Autodesk, HP, Adobe, Ultimaker, Chaos Group ve Microsoft yetkili iş ortağı.

## 📋 Proje Yapısı

```
cadbim_site/
├── index.html              # Ana sayfa
├── assets/
│   ├── css/               # Stil dosyaları
│   ├── img/               # Resimler
│   └── logos/             # Marka logoları
├── cadbim_*.html          # Ürün & hizmet sayfaları
├── sektor_*.html          # Sektör-spesifik sayfalar
└── robots.txt / sitemap.xml
```

## 🚀 Hızlı Başlangıç

### Yerel Ortamda Çalıştır
```bash
# Projeyi klonla
git clone <repository-url>
cd cadbim_site

# Basit bir HTTP server ile aç (Python)
python3 -m http.server 8000

# Veya Node.js ile
npx http-server
```

Tarayıcında açın: `http://localhost:8000`

## 📦 Teknoloji Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Build**: Static site
- **Deployment**: Azure / Vercel / GitHub Pages
